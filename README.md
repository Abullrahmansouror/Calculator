# Python Calculator

A simple command-line calculator built with Python to perform basic arithmetic operations.

## About

This project is a beginner-friendly calculator created using Python. It takes two numbers and a mathematical operator from the user, performs the requested calculation, and displays the result. The calculator runs in a loop, allowing users to perform multiple calculations without restarting the program.

### Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Division-by-zero protection
* Repeat calculations until the user chooses to exit

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Abullrahmansouror/Calculator.git
```

2. Navigate to the project folder:

```bash
cd Calculator
```

3. Run the program:

```bash
python calculator.py
```

> **Note:** Replace `calculator.py` with your Python file name if it is different.

* Using `break` to exit a loop
